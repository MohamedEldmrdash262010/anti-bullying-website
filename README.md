# Anti-Bullying Website

## 📌 Overview

The Anti-Bullying Website is a simple educational web project designed to raise awareness about bullying, its negative effects, and how to prevent it. The project aims to provide students with useful information, interactive content, and support resources in a clean and easy-to-use interface.

## 🚀 Features

* Informational pages about different types of bullying
* Educational content about the effects of bullying
* Interactive quiz to test knowledge
* User feedback / opinion section
* Simple and responsive design (works on mobile and desktop)

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 🎯 Purpose of the Project

This project was created as part of my learning journey in web development to improve my skills and contribute to a positive social message against bullying in schools and online environments.
